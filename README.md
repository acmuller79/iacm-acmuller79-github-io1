# IA-ACMULLER Web - Versão Frontend Only

Versão simplificada que funciona 100% no navegador, sem backend.

## 🚀 Deploy

1. Envie todos os arquivos para o GitHub
2. Conecte na Vercel (sem configuração especial necessária!)
3. Pronto!

## 🔑 Configuração

1. Abra o site
2. Clique em ⚙️ Configurações
3. Cole sua API Key (Gemini ou OpenAI)
4. Comece a usar!

## 📁 Arquivos

- index.html
- styles.css  
- app.js
- README.md

Total: 4 arquivos apenas!
